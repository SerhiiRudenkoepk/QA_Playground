# QA Playground



**QA Playground** is an interactive web platform designed for practicing and learning QA (Quality Assurance) skills, particularly in web and game testing. The project combines theoretical quizzes, API practice, forms, and interactive exercises to help QA enthusiasts develop practical skills.

The site is built with **HTML, CSS, and JavaScript**, and features a modern, responsive design.

Some parts of this projects were made with AI (because I have more knowledge in working with Python than JS and web dev. overall) help for rapid prototyping, but I adapted, implemented and customized the rest manualy. 

---

## 🚀 Features

### Home Page
- Overview of the project and its functionalities  
- Navigation buttons:
  - **[QA Quiz](quiz.html)** — interactive quiz with Junior and Middle level questions  
  - **[CATS AND DOGS API](https://github.com/SerhiiRudenkoepk/API-Cats-Dogs)** — practice real REST API requests  
  - **[About Me](about.html)** — developer bio and links  
  - **[Roadmap](roadmap.html)** — planned modules and future enhancements  

### QA Quiz
- Level selection: Junior / Middle, Software / GameDev  
- 20 random questions per quiz session (from JSON)  
- 30-second timer per question  
- Progress bar showing quiz completion  
- Immediate feedback on correct/incorrect answers  
- Score calculation at the end  
- “Back to Level Selection” button  

### About Me
- Short bio of the developer  
- Links to [GitHub](https://github.com/SerhiiRudenkoepk) and [LinkedIn](https://linkedin.com)  
- Modern glass-style container with animated background  

### CATS AND DOGS API
- Link to "CATS AND DOGS" API

## 🎯 Purpose
QA Playground is designed for both beginners and intermediate testers to:  
- Practice QA concepts interactively  
- Understand API testing and JSON validation  
- Improve skills in bug reporting, form validation, and boundary testing  
- Gain hands-on experience with web application testing scenarios

## 📂 How to Use
1. Clone the repository or open the live GitHub Pages link:  
https://github.com/SerhiiRudenkoepk/QA_Playground.git
